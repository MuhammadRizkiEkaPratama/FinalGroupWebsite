Projects

TRASHTRECK

Sustainify is a mobile application prototype designed in Figma that focuses on improving waste management through a reward-based system. The app works like a trash pickup service similar to ride hailing apps where users can request trash collection and earn rewards for every contribution they make.

MECHANALYTICS

Mechanalytics is a desktop application built using Java and Scene Builder that helps monitor and analyze machine conditions in a factory environment. The app provides insights into machine performance, making it easier to detect potential issues, identify when maintenance is needed, and prevent unexpected breakdowns.

The website helps students discover volunteering opportunities that align with their interests and values, making it easier to get involved in meaningful social and environmental initiatives.

SENTRA

Sentra is a Management Information System (MIS) web application designed for university leaders such as rectors, deans, or campus management. The system monitors lecturer performance based on student feedback and integrates an LLM-powered feature to automatically generate performance summaries.
